## 통계/집계 기능

### 소환사 통계

- [ ] 특정 소환사의 전체 게임 수
- [ ] 총 KDA (킬/데스/어시스트 합계 및 평균)
- [ ] 총 피해량 / 평균 피해량
- [ ] 총 골드 / 평균 골드
- [ ] 승률 (승리 게임 수 / 전체 게임 수)
- [ ] 가장 많이 플레이한 챔피언 TOP 3
- [ ] 라인별 게임 수

### 챔피언 통계

- [ ] 특정 챔피언의 전체 게임 수
- [ ] 챔피언별 평균 KDA
- [ ] 챔피언별 평균 피해량/골드
- [ ] 챔피언별 승률
- [ ] 해당 챔피언을 가장 많이 플레이한 소환사

### 라인 통계

- [ ] 라인별 평균 KDA
- [ ] 라인별 평균 피해량
- [ ] 라인별 승률

### 전체 통계 (대시보드)

- [ ] 전체 게임 수
- [ ] 전체 승률
- [ ] 가장 많이 등장한 챔피언 TOP 10
- [ ] 가장 많이 플레이한 소환사 TOP 10
